Есть таблица wonderful_item с полями:
•  id,
•  title,
•  description
Нужно реализовать API в виде приложения на Yii2, которое имело бы следующие вызовы:
1) добавление нового объекта,
2) получение списка объектов,
3) получение информации об одном объекте,
4) удаление объекта
Доступ к API должен осуществляться по токену. Значение токена может храниться в конфигах. Токен должен идти в http-заголовке X-TESTAPI-TOKEN, а его проверка осуществляться через behaviors.
Формат данных должен быть JSON.
